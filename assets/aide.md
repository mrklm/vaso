# Aide Vaso

## Présentation
Vaso est un générateur de vases polygonaux exportables en STL.

## Onglet Général
Cet onglet permet de :
- régler les paramètres généraux
- définir la forme du vase
- afficher un aperçu 2D
- lancer un aperçu
- générer une nouvelle forme aléatoire
- exporter le STL

## Onglet Options
Cet onglet permet de :
- choisir le dossier d’export STL

Par défaut, les exports sont créés dans un dossier daté sur le Bureau.

## Seed
La seed permet de reproduire exactement une génération aléatoire.

## Export
Chaque export utilise un nom incrémenté automatiquement :
- vaso_export_0.stl
- vaso_export_1.stl
- vaso_export_2.stl

## Thèmes
Le thème peut être changé depuis la combobox en haut à droite.
Un thème aléatoire est sélectionné au démarrage.

## Remarques
Les fichiers STL générés sont destinés à être vérifiés dans un slicer avant impression.
